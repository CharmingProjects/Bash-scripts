# Bash-scripts
## Generators
Download words-gen.sh
```bash
curl https://raw.githubusercontent.com/CharmingProjects/Bash-scripts/master/words-gen.sh --create-dirs -o /usr/local/sbin/words-gen.sh && chmod a+x /usr/local/sbin/words-gen.sh
```
